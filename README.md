## 🚀 Getting Started

### Option 1: View Locally
1. Clone or download this repository
2. Open `index.html` in your web browser
3. Enjoy your interactive resume!

### Option 2: Deploy to GitHub Pages

1. **Update your repository settings:**
   - Go to your GitHub repository → Settings
   - Scroll to "GitHub Pages" section
   - Select "Deploy from a branch"
   - Select "main" branch and "/" (root) as the source
   - Click "Save"

2. **Your site will be available at:**
   ```
   https://lazynoob.github.io/Ravilla-Keerthi-Naidu/
   ```

3. **Update your profile information:**
   - Edit `index.html` to add your actual information
   - Update contact details in the contact section
   - Add your social media links
   - Update project details and links
   - Modify experience and education sections

## ✏️ Customization Guide

### Update Personal Information

Edit the following sections in `index.html`:

**Hero Section:**
```html
<h1 class="hero-title">Your Name</h1>
<p class="hero-subtitle">Your Title</p>
```

**About Section:**
Update the about text and statistics (years experience, projects, satisfaction).

**Experience Section:**
Add your work experience with job title, company, dates, and responsibilities.

**Skills Section:**
List your technical skills organized by category (Frontend, Backend, Tools, Soft Skills).

**Education Section:**
Add your educational qualifications and certifications.

**Projects Section:**
Showcase your best projects with descriptions, technologies used, and links.

**Contact Section:**
Update your contact information:
- Email
- LinkedIn profile
- GitHub profile
- Twitter/Social media

### Color Customization

Edit the CSS variables in `styles.css`:

```css
:root {
    --primary-color: #667eea;
    --secondary-color: #764ba2;
    --accent-color: #f093fb;
    /* ... other colors ... */
}
```

### Font Customization

Update the font family in `styles.css`:

```css
body {
    font-family: 'Your Font Name', sans-serif;
}
```
